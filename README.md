
# Supply Chain Management Database Project

## Project Overview
Designed and implemented a relational database for supply chain operations including suppliers, products, inventory, orders, and shipments.

## Business Problem
Companies need efficient tracking of inventory, suppliers, purchase orders, and deliveries to reduce stockouts and optimize operations.

## Technologies Used
- MySQL
- SQL
- GitHub

## Database Features
- Inventory Management
- Supplier Management
- Purchase Orders
- Customer Orders
- Shipment Tracking


## SQL Concepts Used
- Joins
- Views
- Stored Procedures
- Triggers
- Aggregate Functions
- Indexes
